## ⚠️ Regras de Entrega de Atividades

Para que o seu trabalho seja corrigido, o arquivo Python (`.py`) ou o diagrama (`.drawio`) **deve** ser salvo com o seguinte padrão:

`[NOME]_[SOBRENOME]_[NOME_DA_ATIVIDADE]`

**Exemplo:** `Joao_Silva_FichaRPG.py`

Arquivos enviados como `teste.py` ou `meucodigo.py` não serão avaliados.
