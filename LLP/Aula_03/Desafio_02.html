2. Operadores de Comparação ⚖️

Para tomar decisões, o computador precisa COMPARAR informações. Ele só responde True (Verdadeiro) ou False (Falso).

Símbolo Significado Exemplo Real
==Exatamente Igualsenha == "1234"
!=Diferente (Não Igual)nome != "Hacker"
> ou <Maior / Menorvida > 0
>= ou <=Maior Igual / Menor Igualidade >= 18

⚠️ ATENÇÃO: Um igual (=) GUARDA. Dois iguais (==) COMPARA!

3. Operadores Lógicos 🛡️

Às vezes uma condição só não basta. Precisamos combinar regras.

  • and (E): Rígido. TUDO tem que ser verdade.
    Ex: idade >= 18 and tem_ingresso == True
  • or (OU): Flexível. Pelo menos UM tem que ser verdade.
    Ex: dia == "Sábado" or dia == "Domingo"
  • not (NÃO): Contrário. Inverte a verdade.
    Ex: not chovendo (Se não estiver chovendo...)

4. Indentação 📏

O Python precisa saber quais comandos pertencem a quais decisões. Ele faz isso através do (TAB).


# O comando de baixo só acontece SE o de cima for verdade
if chovendo == True:
    # Esse espaço (TAB) significa que o print está DENTRO do if
    print("Pegue o guarda-chuva!")

print("Vou sair de casa.") # Sem espaço = Fora do if. Vai rodar sempre.

A Regra: Toda vez que tiver : (dois pontos), a próxima linha DEVE ter um Tab (espaço).

5. Condicional (If, Elif, Else) 🔀

É assim que o computador pensa, lê-se de cima para baixo:

  • if (SE): A primeira pergunta. Plano A
  • elif (SENÃO, SE): O plano B. Só testa se o if falhar.
  • else (SENÃO): O plano Z. Se tudo der errado, faça isso (Não leva condição!).

idade = 15

if idade >= 18:
    print("Bem-vindo ao Bar!")
elif idade >= 16:
    print("Você pode entrar, mas só bebe suco.")
else:
    print("Volte para casa.")

⚔️ PAINEL DE MISSÕES: CAÇA AOS BUGS

🟢 Nível 1: Limite de Nível

O jogador pode digitar que é Nível 1000 ou -5.

Use if/else e operadores lógicos para garantir que o nível fique apenas entre 1 e 100. Se ele digitar fora disso, dê uma bronca e defina o nível dele como 1.

🔵 Nível 2: O Caloteiro

A loja vende fiado! Envolva o cálculo da poção com um if/else.

O herói só pode comprar e subtrair o dinheiro SE o ouro dele for maior ou igual ao custo total. Senão, recuse a venda!

🟡 Nível 3: O Hacker das Poções

E se o herói digitar que quer comprar "-10 poções"? A matemática vai somar dinheiro para ele!

Use um if antes da loja para verificar se a quantidade de poções é maior que 0.

🔴 Nível 4: O Combate Final

No final do código, imprima: "Um monstro apareceu! 1-Atacar, 2-Magia, 3-Fugir".

Use If/Elif/Else. Se 1: Dano. Se 2: Use um if interno! Só lança magia se a Mana >= 20. Se 3: Foge. Else: Ação inválida.
